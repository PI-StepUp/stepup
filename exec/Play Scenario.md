# 기능 소개

## 1. 랜플댄(랜덤 플레이 댄스)

### 1-1. 랜플댄 목록

![ranpldan1.gif](/exec/images/randomDance/ranpldan1.gif)

- 랜플댄 목록은 헤더의 ‘랜덤플레이’ 버튼을 통해 이동할 수 있으며, 다음 3가지 항목으로 구성됩니다.
    - 현재 인기 있는 랜덤 플레이 댄스
    - 현재 진행 중인 랜덤 플레이 댄스
    - 진행 예정된 랜덤 플레이 댄스
- 랜플댄 목록은 다음 3가지 항목으로 나누어 확인할 수 있습니다.
    - 현재 인기 있는 랜덤 플레이 댄스
    - 현재 진행 중인 랜덤 플레이 댄스
    - 진행 예정된 랜덤 플레이 댄스

![ranpldan_remocon.gif](/exec/images/randomDance/ranpldan_remocon.gif)

- 우측의 리모컨을 통해 각 항목으로 이동할 수 있습니다.

### 1-2. 랜플댄 개최

![ranpldan_create1.gif](/exec/images/randomDance/ranpldan_create1.gif)

- 랜플댄 목록 우측 상단의 ‘개최하기’ 버튼을 통해 랜플댄을 개최할 수 있습니다.

![ranpldan_create2.gif](/exec/images/randomDance/ranpldan_create2.gif)

[시작 시간이 현재 시간보다 이후인 경우]

![ranpldan_create3.gif](/exec/images/randomDance/ranpldan_create3.gif)

[시작 시간이 현재 시간인 경우]

- 랜플댄 시작 시간을 현재 시간 이후로 설정할 경우, 랜플댄 목록 페이지로 이동합니다.
- 랜플댄 시작 시간을 현재 시간으로 설정할 경우, 호스트 페이지로 이동합니다.

### 1-3. 랜플댄 예약

![ranpldan_reserve2.gif](/exec/images/randomDance/ranpldan_reserve2.gif)

- 시작 전 랜플댄의 경우 ‘예약하기’ 버튼이 활성화 되며, 해당 버튼을 클릭하면 예약이 이루어집니다.

![ranpldan_reserve1.gif](/exec/images/randomDance/ranpldan_reserve1.gif)

- 예약한 랜플댄은 ‘마이페이지’에서 확인할 수 있습니다.

![ranpldan_reserve3.gif](/exec/images/randomDance/ranpldan_reserve3.gif)

- ‘예약 취소’ 버튼을 통해 예약을 취소할 수 있습니다.

### 1-4. 랜플댄 참여

![ranpldan_attend1.gif](/exec/images/randomDance/ranpldan_attend1.gif)

[호스트 방 입장]

![ranpldan_attend2.gif](/exec/images/randomDance/ranpldan_attend2.gif)

[참여자 방 입장]

- ‘참여하기’ 버튼을 통해 랜플댄에 참여할 수 있으며, 호스트의 경우 호스트 방, 참여자의 경우 참여자의 방으로 이동합니다.

![ranpldan_attend3.gif](/exec/images/randomDance/ranpldan_attend3.gif)

- 다른 인원이 들어올 경우 상대 방의 웹 캠이 렌더링 됩니다.

![ranpldan_attend4.gif](/exec/images/randomDance/ranpldan_attend4.gif)

- 좌우 반전, 마이크 On/Off, 카메라 On/Off 기능을 제공합니다.

### 1-5. 랜플댄 진행

![ranpldan_attend5.gif](/exec/images/randomDance/ranpldan_attend5.gif)

![ranpldan_attend6.gif](/exec/images/randomDance/ranpldan_attend6.gif)

- 호스트가 노래를 재생할 경우, 참여자 방에서 3 → 2 → 1 카운트 후 해당 노래의 안무 측정이 시작됩니다.
- 안무 측정이 진행되는 동안 모션 인식을 통해 정답 안무와의 유사도를 측정하며, 해당 유사도를 기반으로 성공/실패 여부가 결정됩니다.

![ranpldan_attend7.gif](/exec/images/randomDance/ranpldan_attend7.gif)
![ranpldan_attend9.gif](/exec/images/randomDance/ranpldan_attend9.gif)

- 안무 측정이 종료되면 결과(SUCCESS, FAILURE)가 렌더링 됩니다.

![ranpldan_attend10.gif](/exec/images/randomDance/ranpldan_attend10.gif)

![ranpldan_attend8.gif](/exec/images/randomDance/ranpldan_attend8.gif)

### 2-4. 게시글 글 작성

**공지사항**

![관리자일 경우](/exec/images/board/공지-%20글작성.gif)

관리자일 경우

![일반 사용자일 경우](/exec/images/board/공지-%20글작성%20admin%20x.gif)

일반 사용자일 경우

- 공지 사항은 관리자에게만 `글 작성하기` 버튼이 활성화 됩니다.
- 소통, 오프라인 정모 게시판에서는 로그인 한 사용자 권한으로 작성이  가능합니다.
- `글 작성하기` 버튼을 통해 글 작성을 할 수 있습니다.

**소통 & 오프라인 정모**

![소통 글등록.gif](/exec/images/board/소통%20글등록.gif)

- `글 작성하기` 버튼을 통해 글 작성을 할 수 있습니다.
- 오프라인 정모일 경우 진행 시간과 날짜, 지역을 기입해 주어야 합니다.
- `글 작성하기` 버튼을 통해 글 작성을 할 수 있습니다.
- 본인이 작성한 게시글은 마이페이지에서도 확인이 가능합니다.

![소통 글수정.gif](/exec/images/board/소통%20글수정.gif)

`글수정` 버튼을 통해 글 수정 페이지로 이동이 가능합니다.

- 작성자만 게시글 수정 버튼이 활성화됩니다 .
    - 제목
    - 내용
    - (오프라인 정모) 시작하는 시간, 끝나는 시작, 지역

### 2-7 게시글 삭제

![게시글 삭제.gif](/exec/images/board/게시글%20삭제.gif)

- 공지사항은 관리자만 게시글 삭제 버튼이 활성화됩니다.
- 소통 및 오프라인 정모는 작성자만 게시글 삭제 버튼이 활성화됩니다.

### 2-8 댓글 등록

![댓글 등록3.gif](/exec/images/board/댓글%20등록3.gif)

- 게시글 상세 목록에서 해당 게시글에 대한 댓글을 등록할 수 있습니다.
- 내용을 입력 후 `댓글등록` 버튼을 누르면 댓글 등록이 완료됩니다.
- 댓글이 등록되면 `댓글이 추가되었습니다.` 알림 창이 뜹니다.
- 댓글은 작성자의 닉네임으로 작성됩니다.

### 2-9 댓글 삭제

![댓글 삭제2.gif](/exec/images/board/댓글%20삭제2.gif)

- 작성자만 `댓글삭제` 버튼이 활성화되고 버튼을 누르면 삭제가 가능합니다.
- 댓글이 삭제되면 `댓글이 삭제 되었습니다.` 알림 창이 뜹니다.

## 4. 연습실

### 4-1. 연습실 입장

![비회원](/exec/images/practice/연습실입장-비로그인.gif)

비회원

![회원](/exec/images/practice/연습실입장-로그인.gif)

회원

- 로그인 하지 않은 사용자는 연습실에 입장 할 수 없습니다.
- 로그인 한 사용자는 연습실에 입장 가능합니다.

### 4-2. 연습실 노래 선택

![노래 선택.gif](/exec/images/practice/노래선택.gif)

- 플레이 리스트에서 원하는 곡을 선택하면 해당 곡의 안무 영상이 나옵니다.

### 4-3. 안무 연습

4-3-1. 안무 영상 플레이

![연습실-롤백.gif](/exec/images/practice/연습실-롤백.gif)

- 좌측 동영상의 `재생 버튼`을 클릭하여 안무를 확인하며 연습 할 수 있습니다.
- 재생 바가 시작 위치에 있지 않아도 `퍼펙트 스코어` 버튼을 클릭하여 측정을 시작하면 영상이 처음부터 시작됩니다.

4-3-2. 좌우 반전

![좌우반전.gif](/exec/images/practice/좌우반전.gif)

- `좌우 반전 버튼`을 클릭하면 웹캠 화면의 좌우가 반전됩니다.

4-3-3. 연습 점수 측정

![연습실-국민체조.gif](/exec/images/practice/연습실-국민체조.gif)

- `퍼펙트 스코어` 버튼 클릭 시, 카운트 다운이 끝나면 안무 영상이 처음부터 재생되며 점수 측정이 시작됩니다.
- 영상이 끝나면 점수 측정이 완료되고 우측 상단의 안무 정확도가 갱신됩니다.
- 영상이 끝나면 점수에 따라 결과가 화면에 송출됩니다.
- 점수 별 연습 결과는 다음과 같습니다.
    - 90점 이상 : `perfect`
    - 80~89점 사이: `great`
    - 70~79점 사이 : `good`
    - 60~69점 사이 : `so so`
    - 50~59점 사이 : `bad`
    - 49점 이하 : `fail`
    

### 4-4. 연습실 나가기

![연습실-나가기.gif](/exec/images/practice/연습실-나가기.gif)

- 좌측 상단 화살표 버튼을 통해 연습실을 나갈 수 있습니다.

## 5. 계정 관리

### 5-1. 회원가입

![회원가입.gif](/exec/images/user/회원가입.gif)

- 아이디, 닉네임, 이메일은 중복 확인 과정을 통해 다른 유저와 동일한 정보를 입력할 수 없습니다.
- 캘린더를 활용하여 생년월일을, 옵션 창을 활용하여 국가를 선택할 수 있습니다.
- 회원가입 이후에는 바로 로그인 창으로 이동하여 사용자의 편의성을 높였습니다.

### 5-2. 로그인 및 로그아웃

![로그인로그아웃.gif](/exec/images/user/로그인로그아웃.gif)

- 가입한 아이디와 비밀번호로 로그인을 시도하면 정상적으로 로그인을 할 수 있습니다.
- 올바르지 않은 정보를 입력하면 "아이디 또는 비밀번호가 틀렸습니다.”라는 창으로 알려줍니다.

### 5-3. 마이페이지 조회

![마이페이지.gif](/exec/images/user/마이페이지.gif)

- 마이페이지에 접속하면 아래 내용을 확인할 수 있습니다.
    - 회원 프로필 사진, 현재 포인트 및 등급, 닉네임, 포인트 적립 내역
    - 예약된 랜플댄 개수 및 목록
        
        → 예약 랜플댄 취소 가능
        
    - 개최한 랜플댄
        
        → 모달창을 활용하여 정보 수정 가능
        
    - 참여한 랜플댄
    - 작성한 게시글(오프라인 정모, 자유게시판) 개수 및 목록
        
        → 해당 게시글로 이동 가능
        
    - 회원정보수정페이지로 이동할 수 있는 비밀번호 입력 창
- 만약, 로그인을 하지 않고 마이페이지에 접속하는 경우, 로그인 창으로 자동으로 이동하게 됩니다.

### 5-4. 회원정보 수정

![회원정보수정.gif](/exec/images/user/회원정보수정.gif)

- 마이페이지 내부의 비밀번호 입력창을 통해 회원정보 수정페이지로 이동할 수 있습니다.
- 수정페이지 내부에서는 아래 정보를 수정할 수 있으며 닉네임, 이메일의 경우 중복 확인 과정을 통해 다른 유저와 동일 정보를 입력할 수 없습니다.
    - 회원 프로필 이미지
    - 닉네임
    - 이메일
    - 국가
    - 이메일 수신 동의
- 수정하고자 하는 정보를 입력한 후, 저장 버튼을 누르면 해당 정보가 저장됩니다.
- 만약 취소 버튼을 누르는 경우, 마이페이지로 이동합니다.
- 만약, 비밀번호를 입력하지 않고 회원정보 수정페이지로 접근하는 경우, 마이페이지로 자동 이동됩니다.

### 5-5. 비밀번호 수정

![비밀번호수정.gif](/exec/images/user/비밀번호수정.gif)

- 마이페이지 내부의 비밀번호 입력창을 통해 회원정보 수정페이지로 이동, 비밀번호를 수정할 수 있습니다.
- 바꾸고자하는 비밀번호를 입력하고, 재입력을 통해 더블 체크를 진행합니다.
    
    입력된 두 비밀번호가 일치하는 경우에만 비밀번호를 수정할 수 있습니다.
    
- 비밀번호가 정상적으로 수정되었다면 메인페이지로 이동합니다.
